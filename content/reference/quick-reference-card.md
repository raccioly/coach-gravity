# 📇 Quick Reference Card

> **Print this page.** Keep it next to your monitor for your first few sessions.

---

## 🗣️ Essential Phrases

| When You Need To... | Say This |
|---|---|
| **Start something** | *"I want to build..."* |
| **Approve a plan** | *"Proceed"* or *"Go ahead"* |
| **Stop the AI** | *"Stop. Don't make any changes."* |
| **Redirect** | *"That's not what I meant. I want X, not Y."* |
| **Simplify** | *"Too complex. Do the minimum."* |
| **Undo** | *"Undo the last change"* |
| **Get explanation** | *"Explain that in simple terms."* |
| **See options** | *"Show me 2-3 options."* |
| **Fix a visual issue** | *[Paste a screenshot]* + describe what's wrong |
| **Check quality** | *"Run /preflight"* |

---

## ⚡ Key Commands

| Command | What It Does |
|---|---|
| `/start` | Guided setup for new users |
| `/launch` | Start your app in the browser |
| `/preflight` | Run quality checks |
| `/sentinel` | Run security scan |
| `/architect` | Get architecture review & score |
| `/courier` | Commit + push + deploy |
| `/stage` | Automated browser testing |

---

## 🖼️ Screenshots = Your Superpower

| To do this... | Do this... |
|---|---|
| **Take screenshot (Windows)** | `Win + Shift + S` → select area → `Ctrl + V` in chat |
| **Take screenshot (Mac)** | `Cmd + Shift + 4` → select area → drag into chat |
| **Show what's broken** | Screenshot it + *"Fix this"* |
| **Copy a design you like** | Screenshot another website + *"Make mine look like this"* |

---

## 🛡️ Safety Guarantees

- ✅ AI shows a plan before every change
- ✅ You approve before anything happens
- ✅ More than 3 files? AI stops and asks first
- ✅ Commands require your permission
- ✅ "Undo" always works

> **Nothing you type can break your computer.** The worst outcome is you say "undo."

---

## 🆘 If Something Goes Wrong

1. Say *"Undo the last change"*
2. Press `F12` in the browser → screenshot any red errors → share
3. Say *"Something broke. Here's what I see: [screenshot]"*
4. If stuck: start a new conversation in the same project folder
